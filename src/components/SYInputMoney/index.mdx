## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| placeholder | 输入提示 | string  | '请输入' |
| disabled | 是否禁用 | bool | false |
| width | 宽度 | Number、String | '80%' |
| precision | 数值精度 | number | 2 |
| onChange | 变化回调 | (val): void |  |
