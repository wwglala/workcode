## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| title | 标题 | string  | '类别' |
| visible | 是否可以切换选中项 | bool | false |
| width | model宽度 | Number、String | 800 、'80%' |
| footer | 固定条里面的内容 | ReactNode | - |
| destroyOnClose | 关闭时销毁 Modal 里的子元素 | bool | false |
| handleOk | 选中确定按钮触发事件 | 	(val): void |  |
| handleCancel | 选中取消按钮触发事件 | 	(val): void |  |
