# SYPersonnelSelection

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| type | 选择控制: 'all','dept','person' | String | all |
| deptName	| 部门文案 | String | 部门名称 |
| personName | 人员文案 | String | 参会人 |
| dataSource | 部门数据源 | [] | - |
| dataSourcePerson	|人员数据源	| [] | - |
| onClick | 部门点击添加触发事件 | () => {} | - |
| onClickPerson | 人员点击添加触发事件 | () => {} | - |
| CallbackData,CallbackDataPerson | 点击删除执行的回调并返回数据 | (data) => {} | - |

- 添加按钮功能为其他业务组件
