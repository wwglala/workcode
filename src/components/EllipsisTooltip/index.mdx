# EllipsisTooltip 

- 超出父组件宽度自动显示为省略号，鼠标移入显示提示信息。

## 在下拉框中使用
<Playground>
<Select style={{ width:'200px' }}>
    <Option key='aaa'>
        <EllipsisTooltip 
        clickClose 
        title='aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa'>
        aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa
        </EllipsisTooltip>
    </Option>
    <Option key='bbb'>bbb</Option>
</Select>
</Playground>

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| title | 提示信息 | reactNode | - |
| clickClose | 点击后隐藏提示 | bool | false |
